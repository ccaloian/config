# nvim config

This configuration is strongly inspired from [this](https://www.youtube.com/playlist?list=PLsz00TDipIffreIaUNk64KxTIkQaGguqn) youtube playlist by [typecraft](https://www.youtube.com/@typecraft_dev) and [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim).

> **IMPORTANT:** `nvim` greater than `v0.9.0` is required. On a Mac, you can install `nvim` using `brew install neovim`.

## Installation

Clone this repository to `~/.config/nvim`:

```bash
git clone <REPO_URL> ~/.config/nvim
```

If you already have a `~/.config/nvim` directory and you want to try this configuration in parallel, you can clone the repo to some other location, _e.g._ `~/temp/nvim-test`, create a symlink in `~/.config/nvim-test` (or clone it here directly) and add the following to your `~/.zshrc` or `~/.bashrc`:

```bash
alias nvt="NVIM_APPNAME=nvim-test nvim"
```

This will allow you to start `nvim` with the test configuration by running `nvt`.
